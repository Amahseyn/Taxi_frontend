# Colchester Airport Taxi - Frontend

Next.js frontend for taxi booking system with animated hero section, booking form, and admin dashboard.

## Features

- Responsive hero section with animated gradient background
- Interactive booking form with vehicle selection
- Admin dashboard for managing bookings and drivers
- User portal for viewing and managing bookings
- Areas map with markers
- Vehicle fleet showcase

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Environment

The frontend expects the backend API at `http://localhost:8000/api/v1`.

## Deployment

Deploy to Vercel:

```bash
npm run build
npm run start
```
